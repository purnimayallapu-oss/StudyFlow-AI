# 🚀 StudyFlow AI

StudyFlow AI is an AI-powered study planning and productivity platform designed to help students not only create study schedules but successfully follow them.

Unlike traditional study planners, StudyFlow AI combines intelligent planning, adaptive rescheduling, spaced repetition, gamification, cognitive refresh games, and AI-powered learning reinforcement to improve consistency, retention, and exam readiness.

---

## 🌟 Why StudyFlow AI?

Most study planner applications stop at creating schedules.

StudyFlow AI helps students:

* Create realistic study plans
* Stay consistent through adaptive scheduling
* Reinforce learning using AI-generated quizzes
* Improve retention with spaced repetition
* Prevent burnout using cognitive mini-games
* Track progress with analytics and streaks
* Receive personalized AI recommendations

### Learning Flow

Exam → Subject → Chapter → Topic → Task

Complete Topic
↓
Earn XP
↓
Play Cognitive Refresh Game
↓
Answer AI Knowledge Check
↓
Receive AI Summary
↓
Continue to Next Topic

Interactive mini-games inspired by LinkedIn Games and modern learning platforms.

Games include:

* Zip Connect
* Crossword Puzzle
* Memory Match
* Pattern Recognition
* Logic Puzzle
* Word Builder
* Flash Recall Challenge

These games refresh the mind without distracting from learning.

## 🧠 AI Learning Reinforcement

After every completed topic:

* Generate 5 AI-powered questions
* Evaluate understanding instantly
* Identify weak concepts
* Generate a concise summary
* Schedule intelligent revisions

The goal is not just completion but actual understanding and retention.

## 🚀 Run Locally

### Prerequisites

* Node.js

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd studyflow-ai
```

2. Install dependencies

```bash
npm install
```

3. Create `.env.local`

```env
GEMINI_API_KEY=your_api_key_here
```

4. Start development server

```bash
npm run dev
```
